# Uni Project - UniStay

## Prerequisites
- Node.js (version 16 or higher)
- MariaDB installed and running

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```
2. Install Dependencies
   ```bash
   npm install
   ```
3. Start the server
   ```bash
   npm start
   ```
